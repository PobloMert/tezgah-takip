# GitHub Bağlayıcısı Test Raporu

Bu rapor, GitHub bağlayıcısının (GitHub CLI aracılığıyla) `PobloMert/TezgahTakip` deposu üzerindeki yeteneklerini göstermektedir.

## GitHub Bağlayıcısının Kabiliyetleri

GitHub bağlayıcısı, GitHub CLI (Komut Satırı Arayüzü) kullanılarak GitHub depolarıyla etkileşim kurma yeteneği sağlar. Bu, aşağıdaki gibi çeşitli işlemleri otomatikleştirmeye ve gerçekleştirmeye olanak tanır:

*   **Depo Yönetimi:** Depo oluşturma, görüntüleme, düzenleme ve silme.
*   **Issue ve Pull Request Yönetimi:** Issue ve Pull Request'leri listeleme, oluşturma, düzenleme ve kapatma.
*   **Commit ve Branch Yönetimi:** Commit geçmişini görüntüleme, branch'leri listeleme ve yönetme.
*   **GitHub API Erişimi:** Doğrudan GitHub REST API'sine erişim sağlayarak daha gelişmiş sorgular ve işlemler yapma.
*   **Otomasyon:** CI/CD süreçlerine entegrasyon ve komut dosyaları aracılığıyla tekrarlayan görevleri otomatikleştirme.
